# QR Code Generator (Python)

A simple Python script that generates QR codes from user-provided URLs using the Python `qrcode` library. This project demonstrates how to use an external Python package to create QR code images.

## About the `qrcode` Library

This project uses the `qrcode` library, a popular Python package for generating QR codes with customizable size, error correction, and colors.

### Official documentation:  
https://pypi.org/project/qrcode/

## Features

- Generate QR codes from any URL
- Custom output file name
- Uses PIL for image generation
- Simple and easy-to-understand code


## Requirements

- Python 3.7 or higher
- `qrcode[pil]`

## Installation

Install the required dependency:

```bash
pip install qrcode[pil]
```
## Usage
Run the script:

```bash
python qrGenerator.py
```
